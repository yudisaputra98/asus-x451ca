- Bootloader Opencore 0.6.3
- Processor Core i3 3217U (IvyBridge)
- VGA Intel HD Graphics 4000

- Ethernet Realtek RTL81xE
- Touchpad ELAN
- WLAN Qualcomm Atheros AR9485
